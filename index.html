📡

Multi-Asset Data

Integrates stocks (NSE/NYSE), gold futures, silver, forex pairs, and financial news via yfinance. 20+ years of daily OHLCV data cached locally.

⚡

40+ Features

RSI, MACD, Bollinger Bands, OBV, rolling volatility, EMA/SMA crossovers, cross-asset correlations, and log-return statistics.

🔮

HMM Regime Detection

Hidden Markov Model identifies Bull, Bear, and High-Volatility market regimes from latent return and volatility patterns.

🤖

3 ML Classifiers

Random Forest, Gradient Boosting, and Logistic Regression trained on a chronological split. No shuffling — no look-ahead bias.

💡

SHAP Explainability

Every prediction is explained using SHapley Additive Explanations — showing exactly which features drove each forecast.

📈

Portfolio Backtester

Long-only strategy simulator computing Sharpe, Sortino, CAGR, max drawdown, and win rate versus a buy-and-hold benchmark.

This project integrates classical machine learning, hidden Markov models, and modern explainability tools to build a transparent financial forecasting system. Unlike black-box deep learning approaches, every prediction can be traced back to specific market signals.

Literature References
Reference 01
Lopez de Prado (2018) — Advances in Financial Machine Learning, Wiley. Feature engineering and cross-validation for financial time series.
Reference 02
Hamilton (1989) — A New Approach to the Economic Analysis of Nonstationary Time Series. Econometrica 57(2). Introduced HMM regime switching.
Reference 03
Breiman (2001) — Random Forests. Machine Learning 45(1). Ensemble learning for high-dimensional classification.
Reference 04
Lundberg & Lee (2017) — A Unified Approach to Interpreting Model Predictions. NeurIPS. SHAP values for consistent feature attribution.
Project File Structure
Module overview
config.py
Central configuration — tickers, dates, hyperparameters, paths.
main.py
CLI runner — executes all 6 pipeline steps in sequence.
app.py
Flask web server — API endpoints, SSE log streaming, serves this UI.
utils/data_ingestion.py
yfinance downloader with CSV caching and date alignment.
utils/feature_engineering.py
RSI, MACD, BB, OBV, volatility, cross-asset correlations, PCA, target.
utils/backtesting.py
Long-only strategy, Sharpe/Sortino/CAGR/drawdown, equity curve.
models/regime_detection.py
GaussianHMM with auto semantic labelling and regime plot.
models/prediction_models.py
RF, GBM, LR pipelines, SHAP explainability, feature importance.
utils/sentiment_analysis.py
TF-IDF + Naive Bayes for FNSPID news, daily score aggregation.
